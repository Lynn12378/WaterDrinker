# WaterDrinker
# 1.	專案簡介:
記錄使用者與飲水機的互動，並且使用者可以透過Line Bot聊天機器人查看其每日飲水量、設置自動出水模式、開啟每段時間間隔提醒用水等功能。
# 2.	專案目的與動機
此專案希望協助使用者紀錄喝水習慣，並提醒每間隔一段時間提醒使用者飲水，期望達到每日建議飲水量。
